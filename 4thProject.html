PG Life: Find pg's near you

PG Life is a web-based application developed to simplify the process of finding and managing PG (Paying Guest) accommodations for students and working professionals. It allows users to browse through various PG listings, view details, and contact the property owners directly.

The platform features an interactive and responsive UI, login/signup authentication, dynamic listing pages, filtering options (by city, gender, etc.), and individual PG detail pages. The admin panel supports adding or removing PG listings from the backend using PHP and MySQL.

This project helped me strengthen my skills in web development, especially in backend integration using PHP and database handling with MySQL.
